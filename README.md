# 🏁 Syncro Motos - Software de Gestión para Talleres

¡Bienvenido al instalador oficial de **Syncro Motos**! Esta aplicación está diseñada para optimizar la gestión de tu taller de motos, incluyendo inventarios, CRM, facturación electrónica y más.

## 📥 Descarga la Aplicación

Puedes descargar la versión más reciente directamente desde este repositorio:

### 🪟 Para Windows
Descarga el archivo: **`Syncro Motos Setup 1.0.0.exe`**
1. Ejecuta el instalador.
2. Sigue los pasos en pantalla.
3. La aplicación se abrirá automáticamente.

### 🍏 Para Mac (Apple Silicon M1/M2/M3)
Descarga el archivo: **`Syncro Motos-1.0.0-arm64.dmg`**
1. Abre el archivo `.dmg`.
2. Arrastra el icono de **Syncro Motos** a tu carpeta de **Aplicaciones**.
3. Abre la aplicación desde tu Launchpad.

## ✨ Características Principales
- ✅ **Gestión de Inventario Inteligente**: Control total de repuestos y stock.
- 💬 **Integración con WhatsApp**: Comunicación directa con clientes.
- 🧾 **Facturación Electrónica**: Integración con Factus.
- 🤖 **Asistente AI**: Ayuda inteligente para la toma de decisiones en el taller.
- 📊 **Dashboards en Tiempo Real**: Visualiza el estado de tu negocio.
