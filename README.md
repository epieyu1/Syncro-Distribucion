# 📦 Syncro Motos - Distribución y Updates

Este repositorio actúa como el canal central de distribución para la aplicación **Syncro Motos Desktop**. 

## 🚀 Propósito
- Albergar los archivos de metadatos (`latest.yml`) para el sistema de **Auto-Update**.
- Servir como punto de anclaje para los **GitHub Releases**.

## 🛠️ Gestión de Versiones
Los binarios (`.exe`, `.dmg`) **NO** se almacenan en este repositorio para evitar el inflado del árbol de Git. 
- Los instaladores se cargan manualmente en la sección de **Releases**.
- El sistema de actualización de la aplicación consulta este repositorio para detectar nuevas versiones.
